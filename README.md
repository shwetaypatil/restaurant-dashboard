# 🍽️ RestoManager – Restaurant Dashboard (Front-End Assignment)

## 📌 Overview

RestoManager is a customized **restaurant-focused dashboard** built as part of a Front-End Developer Internship assignment for **hellobooks.ai**.

The goal of this project was to transform a **generic accounting MVP** into an **industry-specific solution** tailored for the restaurant business.

This dashboard combines **financial management** with **operational insights**, helping restaurant owners monitor performance and make better decisions.

---

## 🚀 Features

### 📊 Financial Management

* Revenue tracking
* Expense monitoring
* Net profit calculation
* Expense breakdown visualization
* Staff cost tracking

### 🍴 Operational Insights

* Orders today
* Top selling dishes
* Orders by type (Dine-in, Delivery, Takeaway)
* Peak hour analysis
* Inventory alerts

### 🤖 AI Smart Insights

* Predicts peak order hours
* Detects inventory shortages
* Identifies demand trends
* Suggests operational improvements

### ⚡ Quick Actions

* Add Order
* Add Expense
* Add Supplier
* Create Invoice

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **Chart.js**
* **Google Fonts (Inter)**

## 🎯 Purpose of the Project

The original HelloBooks dashboard provides **generic accounting features** such as income, expenses, and reports.

This project enhances it by:

* Adding **restaurant-specific metrics**
* Improving **data visualization**
* Introducing **AI-driven insights**
* Combining **accounting + operations in one dashboard**
